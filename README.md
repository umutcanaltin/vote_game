# Matter Bubbles Starter

A minimal project structure for **React + Matter.js + Tailwind** to build your nightlife genre voting game.

## Structure
```
matter-bubbles-starter/
├── index.html
├── package.json
├── postcss.config.js
├── tailwind.config.js
├── vite.config.js
├── public/
└── src/
    ├── index.css
    ├── main.jsx
    ├── App.jsx
    └── components/
        └── Bubbles.jsx
```

## Run
```bash
npm install
npm run dev
```
